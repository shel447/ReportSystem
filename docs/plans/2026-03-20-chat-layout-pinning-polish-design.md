# Chat Layout Pinning And Width Polish Design

## Goal
进一步收紧聊天页版式：助手面板消息上限下调到工作区 60%，会话记录栏固定在视口内并独立滚动，输入区贴近页面底部只留薄安全边距，分隔线折叠按钮移动到竖线中部并适度放大。

## Decision
- 助手纯文本消息保持当前 `fit-content + 40%`。
- 助手带 action 的消息改为最大 `60%`，不再占用过宽空间。
- 会话记录栏改成真正的视口内固定列，列表自身滚动，不跟消息流上下跑。
- 输入区保持 fixed，但底部间距压到 `8px`，同步收缩消息区底部预留。
- 折叠按钮移到竖线中部，扩大到更容易点击的尺寸。

## Impact
- 助手面板类消息会更克制，不会显得压过消息流。
- 左侧会话记录会像独立导航列一样稳定。
- 输入区会更贴底，底部空白明显减少。
