# 评审记录

按"提出意见 → 处理意见"的生命周期管理所有检视历史。

## 检视流程

1. **(a) 提出意见**：评审报告记录发现的问题和改进建议，状态标记为`待处理`
2. **(b) 处理意见**：逐条确认或驳回建议，在报告中补充处理记录，索引同步更新状态

## 状态说明

| 状态 | 含义 |
|------|------|
| 待处理 | 评审完成，建议尚未逐条确认 |
| 处理中 | 部分建议已采纳或驳回，仍有未处理项 |
| 已闭环 | 所有建议均已处理（采纳/驳回/记录原因） |
| 不处理 | 整份评审不纳入后续工作 |
