# 智能问数

智能问数让用户直接用自然语言查询业务数据。系统会理解问题、查询可用数据，并返回结论、明细和适合的可视化结果。

## 用户可以做什么

- 询问指标、趋势、分布和明细，例如“本月核心设备告警按级别怎么分布？”
- 查看系统生成的自然语言结论。
- 查看本次查询使用的数据和查询语句。
- 查看表格和图表形式的结果。
- 在同一会话中继续追问。

## 数据理解

- 系统会结合可用数据目录理解实体、字段和关联关系。
- 系统可以检索指标解释和历史查询样例，提升查询准确度。
- 查询执行前会检查生成语句的安全性。
- 查询结果会保留字段含义和必要的数据血缘，便于追溯。

## 异常处理

- 用户输入、返回答案或生成的查询存在安全风险时，系统会拒绝继续处理并给出提示。
- 查询服务失败、超时或返回非法数据时，系统会明确提示失败，不把空结果伪装成正常答案。
- 知识检索不可用时，系统仍会尝试完成查询，但结果可能缺少补充解释。

## 与报告生成的关系

智能问数和报告生成共享数据查询、字段解释和可视化能力。智能问数面向即时分析；报告生成面向按模板冻结的正式内容。
