# 报告管理

报告管理用于查看和使用已经生成的正式报告。它不负责识别模板或生成内容，而是让用户方便地找到报告、预览报告正文或页面内容，并下载文档。

## 用户可以做什么

- 查看自己的报告列表。
- 打开报告详情，了解报告来源和当前状态。
- 预览完整报告。
- 查看报告大纲并快速定位章节或页面。
- 打开本地设计器调整展示效果。
- 生成和下载 Word、PPT 或 PDF 文档。

## 预览与编辑

- 报告详情优先展示正式报告的真实预览，而不是原始数据结构。
- 连续阅读型报告按目录和章节浏览。
- 分页汇报型报告按页面浏览，并展示封面、目录、章节页面和封底。
- 预览和本地设计器使用同一份正式报告，编辑后可以立即查看变化。
- 当前设计器修改只保存在本地工作区，不会自动覆盖后台正式报告。

## 来源追溯

- 报告保留来源会话，便于回看生成过程。
- 报告保留使用的模板信息，便于理解生成口径。
- 章节重新生成会返回预览结果，不会静默改写原报告。

## 用户隔离

- 用户只能查看和下载自己的报告。
- 下载文档时，系统会同时校验报告和文档归属。

## 当前限制

- 本地设计器尚未提供保存到后台的能力。
- 报告版本管理和“将章节预览应用到正式报告”的保存流程仍待后续开放。
