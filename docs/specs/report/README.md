# 报告业务

报告业务覆盖从维护报告模板到生成、查看和导出正式报告的完整链路。它是 ReportSystem 的核心业务域；通用对话只提供交互通道，智能问数提供可复用的数据分析能力。

## 业务链路

```text
维护报告模板
  -> 通过对话提出报告需求
  -> 补充生成条件并确认大纲
  -> 生成和预览正式报告
  -> 调整局部内容
  -> 导出 Word、PPT 或 PDF 文档
```

## 子能力

1. [模板管理](template-management.md)：维护可复用的报告模板。
2. [报告生成](generation.md)：通过对话生成报告，并支持单章节重新生成。
3. [报告管理](management.md)：查看、预览和管理已经生成的正式报告。
4. [文档导出](document-export.md)：把正式报告导出为 Word、PPT 或 PDF 文档。

## 与其他业务能力的关系

- [通用对话](../conversation/README.md) 负责会话、追问、答复和实时消息，不定义报告参数和报告内容规则。
- [智能问数](../data-analysis/README.md) 提供数据查询、字段解释和可视化能力，可被报告生成复用。
- 报告模板、生成草稿、正式报告和导出文档都按当前用户的数据隔离规则使用；模板是平台共享配置资产。
