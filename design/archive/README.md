# 归档文档

本目录存放已归档、已被替代或只保留历史输入价值的文档。

当前归档子目录包括：

- [legacy-entrypoints/README.md](legacy-entrypoints/README.md)：历史根目录入口文档

使用规则：

- 归档文档不得再作为当前设计的权威来源。
- 当正式设计被新的主设计包取代时，旧文档应移动到本目录，而不是继续放在主阅读路径中。
- 需要追溯设计演进或原始输入时，再按需查阅。
