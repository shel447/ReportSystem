# 定时任务专题（归档）

> 状态说明：本专题对应的 `scheduled-tasks` 公开接口当前未挂载，不属于当前公开业务面。  
> 本文档仅作为历史设计与后续恢复/重构时的参考，不代表当前可用能力。

---

## 1. 当前口径

- `/rest/chatbi/v1/scheduled-tasks/*` 当前不公开
- 公开业务面以 `templates / chat / reports / parameter-options` 为准
- 若后续恢复定时任务能力，需要先完成：
  - 对外接口契约重审
  - 用户隔离与配额策略复核
  - 报告时间语义与数据时间窗口建模
  - 运行时调度与失败恢复策略设计

---

## 2. 与现有系统的关系

- 数据模型层仍可能保留 `tbl_scheduled_tasks`、`tbl_scheduled_task_executions` 兼容结构
- 这些结构当前不作为公开能力对外承诺
- 报告主链路不依赖定时任务模块即可闭环运行

---

## 3. 恢复该专题时的最小交付

1. 新版 API 设计文档（请求/响应、错误码、限流、权限）
2. 调度执行语义文档（时区、计划时间、重试、幂等）
3. 数据模型迁移文档（表结构与历史数据处理）
4. 端到端验收清单（创建、暂停/恢复、run-now、失败恢复）
